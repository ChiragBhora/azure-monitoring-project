# Cloud Infrastructure Monitoring and Alerting System using Microsoft Azure

## Project Overview
This project implements a cloud infrastructure monitoring and alerting solution using Microsoft Azure services. Azure Monitor and Log Analytics Workspace were integrated with an Azure Virtual Machine to collect, analyze, and visualize performance metrics such as CPU utilization, network traffic, disk activity, and VM availability.

## Services Used
- Azure Virtual Machine
- Azure Monitor
- Log Analytics Workspace
- Azure Alerts
- Azure Workbooks

## Features
- Real-time infrastructure monitoring
- Automated alert generation
- Performance metrics visualization
- Centralized log management
- Dashboard monitoring

## Architecture
Azure VM → Azure Monitor → Log Analytics Workspace → Alerts → Dashboard
